# 부록 A: 핵심 용어 사전

Claude Code와 AI 에이전트를 다루면서 자주 등장하는 용어를 정리했습니다.

---

## ㄱ

**컨텍스트 윈도우 (Context Window)**
AI가 한 번의 요청에서 처리할 수 있는 최대 텍스트 분량. "기억력"의 크기라고 이해하면 됩니다.
Claude Code는 매우 큰 컨텍스트 윈도우를 가집니다.

## ㄴ

**노드제이에스 (Node.js)**
자바스크립트를 브라우저 밖에서 실행할 수 있게 하는 런타임. Claude Code 설치에 필요합니다.

## ㄷ

**디렉토리 (Directory)**
폴더의 기술적 표현. `Working Directory`는 현재 작업 중인 폴더를 의미합니다.

## ㅇ

**에이전트 (Agent)**
단순히 답변하는 것을 넘어, 도구를 사용하고 행동을 실행하는 AI 시스템.
Claude Code는 파일 읽기/쓰기, 명령어 실행 등의 행동을 수행하는 에이전트입니다.

**이터레이티브 리파인먼트 (Iterative Refinement)**
첫 결과를 기반으로 피드백을 주고, 점진적으로 개선해 나가는 작업 방식.
Claude Code와 일하는 핵심 방법론입니다.

## ㅍ

**프롬프트 (Prompt)**
AI에게 보내는 지시문. 좋은 프롬프트는 역할, 맥락, 출력 형식, 제약을 포함합니다.

## ㅎ

**CLAUDE.md**
Claude Code가 프로젝트를 시작할 때 가장 먼저 읽는 마스터 설정 파일.
프로젝트의 목표, 규칙, 스타일, 현재 상태를 기록합니다.
