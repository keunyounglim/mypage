# Day 1: Claude Code란 무엇인가

> **오늘의 한 줄 요약**: Claude Code는 채팅창이 아니라, 당신의 컴퓨터에서 직접 작동하는 AI 에이전트입니다.

---

## 먼저 실행해 보세요

아래 명령어를 터미널에 그대로 복사해서 붙여넣으세요. 설명은 나중에 해도 됩니다.

```bash
claude "현재 폴더에 있는 파일 목록을 알려줘"
```

**실행 후 나타나야 할 결과:**

```
현재 폴더에는 다음 파일들이 있습니다:
- README.md
- CLAUDE.md
...
```

Claude Code가 현재 폴더를 직접 들여다보고 응답했습니다. 채팅창에서는 불가능한 일입니다.

---

## Why — 왜 Claude Code인가

### 기존 AI 채팅의 한계

지금까지 AI를 써봤다면 이런 경험이 있을 겁니다.

- "내 파일 좀 수정해줘" → "파일 내용을 직접 붙여넣어 주세요"
- "이 폴더 정리해줘" → "어떤 파일이 있는지 알려주세요"
- "계속 진행해줘" → (새 대화를 시작하면 이전 맥락이 사라짐)

AI가 아무리 똑똑해도, 당신의 컴퓨터를 **직접 볼 수 없기** 때문입니다.

### Claude Code가 다른 이유

Claude Code는 당신의 **로컬 환경에 직접 연결**됩니다.

| 기존 AI 채팅 | Claude Code |
|------------|-------------|
| 텍스트만 주고받음 | 파일을 직접 읽고, 쓰고, 수정함 |
| 매 대화가 독립적 | 프로젝트 전체를 기억함 |
| "알려주세요"가 필요 | 스스로 파악하고 실행함 |
| 복붙이 필요 | 바로 적용됨 |

---

## What — 핵심 개념 2가지

### 개념 1: 에이전트 vs 채팅

**채팅 AI**는 질문에 답하는 조수입니다.
당신이 모든 정보를 제공해야 하고, 결과를 직접 적용해야 합니다.

**에이전트 AI**는 목표를 받아 스스로 행동하는 파트너입니다.
파일을 읽고, 코드를 실행하고, 결과를 확인하고, 수정하는 과정을 자율적으로 반복합니다.

```
채팅 AI: "여기 코드 있어요" → 당신이 복사 → 당신이 붙여넣기 → 당신이 실행
에이전트: "이 파일 고쳐줘" → Claude가 파일 읽기 → 수정 → 저장까지 완료
```

Claude Code는 에이전트입니다. 지시만 하면 됩니다.

---

### 개념 2: 컨텍스트 윈도우

에이전트가 한 번에 기억할 수 있는 정보량을 **컨텍스트 윈도우**라고 합니다.
Claude Code는 이것이 매우 큽니다.

**예시로 이해하기:**

상상해 보세요. 새 직원이 첫 날 출근했습니다.

- **컨텍스트가 작으면**: 이 직원은 메모장 한 쪽 분량만 기억합니다. 5분 전 대화도 잊을 수 있습니다.
- **컨텍스트가 크면**: 이 직원은 두꺼운 책 한 권을 통째로 기억합니다. 긴 프로젝트 전체를 이해하며 일합니다.

Claude Code는 수만 단어 분량의 코드와 대화를 한 번에 처리합니다.
그래서 "아까 말한 거 기억해?" 같은 말이 필요 없습니다.

**실용적 의미:**

```
일반 채팅: 대화가 길어질수록 초반 내용을 잊음
Claude Code: CLAUDE.md 파일로 컨텍스트를 보존하고 재사용함
```

> CLAUDE.md 작성법은 Day 3에서 자세히 배웁니다.

## How — 오늘의 실습

### 설치 확인

```bash
# Node.js가 설치되어 있는지 확인
node --version

# Claude Code 설치
npm install -g @anthropic-ai/claude-code

# 설치 확인
claude --version
```

### 첫 대화 시작

```bash
# 프로젝트 폴더로 이동
cd 내-프로젝트-폴더

# Claude Code 시작
claude
```

대화창이 열리면 자유롭게 질문해 보세요.

```
> 이 폴더에 어떤 파일이 있는지 알려줘
> 간단한 README 파일을 만들어줘
> 방금 만든 파일을 열어서 내용 확인해줘
```

### 나가기

```bash
exit
# 또는 Ctrl + C
```

## 오늘의 정리

오늘 배운 핵심 3가지:

1. **Claude Code는 에이전트다** — 채팅이 아니라 당신 컴퓨터에서 직접 작동한다
2. **파일을 직접 다룬다** — 읽기, 쓰기, 수정이 바로 일어난다
3. **컨텍스트 윈도우가 크다** — 긴 프로젝트 전체를 기억하며 일할 수 있다
