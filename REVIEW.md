## 발견된 문제점

### 1. [심각] HTML 구조 오류 - `<p class="msg">` 태그 (156~165번 줄)

**문제:**
```html
<!-- 156번 줄 -->
<p class="msg"></p>   <!-- ← 빈 태그로 즉시 닫힘 -->
    서로 다른 두 사람이 만나<br>
    ...
    감사한 마음 깊이 간직하며 잘 살겠습니다.
</p>                  <!-- ← 고아 닫힘 태그 (짝이 없는 </p>) -->
```

**원인:** `<p class="msg"></p>`가 빈 채로 닫혀 있고, 본문 내용이 태그 바깥에 있습니다.
**결과:** 초대 본문 글이 화면에 표시는 되지만, 의도한 `.msg` 스타일(줄간격, 글자간격)이 적용되지 않습니다.

**수정 방법:**
```html
<p class="msg">
    서로 다른 두 사람이 만나<br>
    사랑으로 하나 되는 자리에<br>
    소중한 여러분들을 초대합니다.<br>
    기쁠 때나, 슬플 때나<br>
    서로를 사랑으로 품으며<br>
    행복의 계단을 쌓아가겠습니다.<br>
    저희의 첫 시작을 함께 해주신다면<br>
    감사한 마음 깊이 간직하며 잘 살겠습니다.
</p>
```

---

### 2. [심각] HTML 구조 오류 - 고아 `</p>` 태그 (172번 줄)

**문제:**
```html
<input type="text" placeholder="이름"></p>   <!-- ← </p> 짝 없음 -->
```

**원인:** `<p>` 여는 태그 없이 `</p>` 닫힘 태그만 존재합니다.
**결과:** HTML 유효성 검사 실패. 브라우저가 임의로 구조를 수정합니다.

**수정 방법:**
```html
<input type="text" placeholder="이름">
```

---

### 3. [보통] `lang` 속성 불일치 (2번 줄)

**문제:**
```html
<html lang="en">   <!-- 영어로 설정되어 있으나 내용은 한국어 -->
```

**원인:** 한국어 콘텐츠임에도 언어 속성이 `en`(영어)으로 설정되어 있습니다.
**결과:** 스크린 리더가 잘못된 언어로 발음, 검색엔진 최적화(SEO) 불이익.

**수정 방법:**
```html
<html lang="ko">
```

---

### 4. [보통] 이미지 `alt` 속성 누락 (147번 줄)

**문제:**
```html
<img src="wedding.avif">   <!-- alt 속성 없음 -->
```

**결과:** 이미지 로드 실패 시 대체 텍스트 없음, 웹 접근성 기준(WCAG) 미충족.

**수정 방법:**
```html
<img src="wedding.avif" alt="김영희 & 이철수 웨딩 사진">
```

---

### 5. [보통] 폼 기능 미구현 (170~175번 줄)

**문제:**
```html
<div class="bottom">
    <input type="text" placeholder="이름">
    <textarea placeholder="메시지를 남겨주세요"></textarea>
    <button>메시지 남기기</button>   <!-- ← 클릭해도 아무 동작 없음 -->
</div>
```

**원인:** `<form>` 요소 없음, 버튼에 이벤트 핸들러 없음, 데이터 전송 로직 없음.
**결과:** 메시지 남기기 버튼이 동작하지 않습니다.

**수정 방법 (최소):**
```html
<form class="bottom" onsubmit="return false;">
    <p>축하메시지를 남겨주세요</p>
    <input type="text" placeholder="이름">
    <textarea placeholder="메시지를 남겨주세요"></textarea>
    <button type="submit">메시지 남기기</button>
</form>
```
> 실제 데이터 저장을 위해서는 백엔드 서버 또는 외부 서비스(예: FormSpree) 연동이 필요합니다.

---

### 6. [경미] AVIF 이미지 브라우저 호환성

**문제:** `wedding.avif` 파일 형식은 구형 브라우저(IE, 구형 Safari 등)에서 지원되지 않습니다.

**수정 방법:**
```html
<picture>
    <source srcset="wedding.avif" type="image/avif">
    <img src="wedding.jpg" alt="김영희 & 이철수 웨딩 사진">
</picture>
```
